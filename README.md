# open_annotation
원하는 사각형의 가장 가까운 꼭짓점 근처를 클릭하면 BBox가 생성됩니다. 기존의 사각형을 직접 그려줘야하는 방법보다 단순하고 쉽습니다. 현재는 파이썬 버전으로만 개발을 하고 있으며, MacOS에서 테스트 해보았습니다.

## Dependencies 의존성
- python3 >= 3.6
- OpenCV >= 3.3

## Installation 설치
- OpenCV

`$ pip3 install opencv-python`
- Git clone

`$ git clone https://github.com/hdh7485/open_annotation.git`

## Run 실행
`$ cd open_annotation`

`$ python3 run.py`
